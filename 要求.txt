## 2. 界面与交互设计要求（参考视频剪辑软件 / Video Editor Style）
界面布局类似于 PR/剪映，分为 4 大核心区域：

1. **左上方：多模态/视觉预览区（Player Area）**
   - 显示当前的 RGB 图像和 Depth 深度伪彩色图（若有图像数据）。
   - 随时间轴指针（Scrubber）的拖动，实时刷新对应时间点的图像与传感器实时数值卡片。

2. **下半部分：动态多轨道时间轴区（Multi-track Timeline UI - 核心）**
   - **主轨（Master Track）**：指定某一低频设备（如 RGB-D 相机 30Hz）的时间戳作为主刻度（Master Grid/Keyframes）。
   - **动态辅轨（Dynamic Sub-tracks）**：根据用户配置的传感器列表，把所有高频传感器波形以多条并行折线图（Plotly/Interactive Chart）排开，共享同一个 X 轴（时间轴 $T$）。
   - **播放控制与指针（Scrubber/Playhead）**：一条垂直的红线表示当前时间点，支持鼠标拖拽，拖动时画面与所有传感器波形联动。
   - **区间选择框（Span Selector）**：支持用鼠标直接在波形图上**拖选一个时间范围 $[T_{start}, T_{end}]$**，或者用快捷键设置标记点（按 `I` 键设置 In 点，按 `O` 键设置 Out 点）。

3. **右侧/顶部：操作与控制面板（Control Panel）**
   - **一键对齐（Alignment）**：触发基于主轨的时间戳重采样/插值对齐。
   - **“场记板/踩地”打板对齐（Clapperboard Auto-Align）**：自动检测 IMU/足底力中的物理冲击峰值（如用力踩地），校正不同设备的系统时间零点漂移。
   - **自动变点分割（Auto-Segmentation）**：调用 `ruptures` 算法，根据传感器波形突变自动将数据切分为若干个“同质区间”。
   - **标签管理器（Label Manager）**：给选定的区间一键赋标签（例如：`平地`、`坡道`、`楼梯`、`草地`），并支持记录边界过渡期。
   - **导出按钮**：一键打包导出为标准的训练数据集。

4. **系统配置与 AI 助手面板（Settings & LLM Panel - 新增）**
   - **传感器动态配置（Sensor Schema Editor）**：允许用户在界面上“自由添加/删除/修改设备与数据列”或导入 YAML 配置文件。
   - **LLM API 配置与智能辅助标注（LLM/VLM Assistant）**：支持配置 API Key、Base URL 与 Model Name，可一键调用大模型进行“传感器数据+图片”的自动地形预判与异常分析。

## 3. 核心算法与软件架构逻辑

### 模块 A：灵活的传感器管道（Dynamic Sensor Pipeline）
- **配置文件解耦**：使用 YAML/JSON 管理传感器 Schema。不硬编码任何传感器名字。
- **自定义通道映射**：用户可指定“设备名称”、“文件路径（CSV/图像目录）”、“时间戳列名”、“数值列名”、“采样频率”及“插值策略（最近邻/线性插值/前向填充）”。

### 模块 B：主轨网格对齐（Master-Grid Alignment）
1. 载入所有已配置的设备数据。
2. 以用户设定的主设备时间戳 $T_{master}$ 为主帧（Master Keyframes）。
3. 使用 `pandas.merge_asof` 或线性插值，把不同频率的传感器数值精准映射到主设备的每一个时间点上。

### 模块 C：基于时间连续性的“区间/片段标注”（Interval Annotation）
1. 用户选定时间范围 $[T_1, T_2]$ 后，给该区间赋予属性 `Terrain_Class`。
2. 数据集内部自动将该区间内的所有帧填充上对应的标签。
3. 提供“边界过渡区（Boundary Transition）”选项：可选将区间交界处 $\pm 0.5\text{s}$ 的数据标记为 `Ignore` 或软标签。

### 模块 D：变点检测（CPD）与 LLM 智能辅助标注
1. **算法变点**：使用 `ruptures` 算法对指定的传感器信号做波形变化率分析，自动寻找切分点。
2. **LLM 预判断**：提取某个区间的传感器统计特征（如平均倾角、足底力方差）并搭配对应的视觉图像帧，构造 Prompt 发送给 LLM / VLM API，由大模型返回建议的标签和推理逻辑（例如：*"图像显示有斜坡，且 Z 轴角度增至 12°，建议标注为 [坡道]"*），用户点击确认即可应用。

### 模块 E：统一导出
将对齐且标注完毕的数据，一键打包导出为 PyTorch 友好的格式（如 `.h5` / HDF5 或单张 master `.csv`），并保存当前使用的传感器配置 Schema。
